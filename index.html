This graph shows some of the largest rocky and icy objects of the Solar System. The selection includes all objects with a radius larger than 400 km that have been imaged in high resolution. Mouse over an object to bring it to the front and to display a fact sheet about it. Created by Daniel Eriksson using d3.js. Source: Wikipedia. Images by NASA, except Ganymede for which the image is by NOAA. The code is available on Github.
